    제주국제공항 → 신제주 → 해군제주방어 사령부 →
    난자농업연구소 → 제주대학교 교차로 → 교래리 입구 →
    비자림로(1112) → 교래사거리 → 산굼부리 → 대천통사거리 →
    서성로(1119번 도로) → 성산읍(섭지코지)→ SUNSET RESORT 제주
